# Student Profile Card App (Module 14)

This is a simple Flutter application developed as part of **Module 14 Assignment**.

The app displays a student profile card with shimmer loading, responsive design, custom dialog, and snackbar using Flutter widgets and packages.

## 📱 Features
- Single screen application
- Shimmer loading effect on app start
- Responsive UI using flutter_screenutil
- Student profile card with avatar and badge
- Custom dialog for student details
- SnackBar message for attendance marking

## 🛠️ Packages Used
- flutter_screenutil
- shimmer

## 📂 Project Structure
lib/
- main.dart  
- screen/
  - student_profile_screen.dart  
- widget/
  - profile_card.dart  
  - shimmer_card.dart  
  - details_dialog.dart  

## ▶️ How to Run
1. Clone the repository  
2. Run `flutter pub get`  
3. Run the app using `flutter run`

## 📌 Note
- Only one screen is used as per assignment instruction.
- Widgets are separated for clean and readable code.
